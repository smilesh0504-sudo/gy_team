
# Spendy: AI 페르소나 분석기 - 버전 1.19 요구사항 정의서

이 문서는 'Spendy' 웹 애플리케이션 버전 1.19의 기능적, 비기능적 요구사항을 정의합니다.

---

## 1. 버전 1.19 주요 변경 사항 요약

*   **절약 팁 기능 추가**: 분석 결과 페이지에서 사용자의 페르소나(주 소비 카테고리)에 맞춘 맞춤형 절약 팁 2가지를 제공하여 실질적인 소비 개선을 유도함.
*   **프로젝트 파일 정리**: 루트 디렉토리의 `README.md`를 정리하고, 불필요한 파일 삭제를 안내.

---

## 2. 상세 기능 요구사항

### 2.1. 절약 팁 제공 (Saving Tips)
-   **맞춤형 팁**:
    -   페르소나 분석 완료 후, 결과 페이지의 '카테고리별 지출' 그래프 하단에 **"절약 꿀팁"** 섹션을 추가한다.
    -   해당 섹션은 분석된 주 페르소나에 매핑된 2가지의 구체적인 행동 지침을 포함해야 한다.
    -   **UI/UX**: 눈에 띄는 전구 아이콘(💡)과 따뜻한 색감(노란색/오렌지색 배경)을 사용하여 긍정적인 느낌을 전달한다.

-   **카테고리별 팁 내용**:
    -   **식비**: 식단 계획하기, 텀블러/도시락 이용하기
    -   **쇼핑**: 장바구니 24시간 대기 법칙, 계절별 쇼핑 원칙
    -   **주거**: 대기 전력 차단, 통신비/관리비 할인 혜택 찾기
    -   **교통비**: 알뜰교통카드 활용, 단거리 도보 이용
    -   **문화/여가**: 문화가 있는 날 활용, 미사용 구독 서비스 해지
    -   **생활비**: 대용량/할인 구매, 지역화폐 활용
    -   **알 수 없음**: 꼼꼼한 기록 습관, 카드 사용 권장

### 2.2. 파일 정리 (Cleanup)
-   **README 정리**: 루트 경로의 `README.md`는 상세 내용을 제거하고, `docs/versions` 폴더를 참조하도록 안내하는 인덱스 역할만 수행한다.
-   **미사용 파일**: `hooks/useGeneratedIcons.ts` 등 더 이상 참조되지 않는 파일은 삭제 권고 대상이다.

---

## 3. 비기능 요구사항

-   **확장성**: 향후 팁 데이터를 서버에서 가져오거나 AI로 생성할 수 있도록 `Persona` 인터페이스에 `tips` 배열 필드를 추가하여 구조를 확장성 있게 설계한다.
